# Response Formatting Templates

## Guidelines
- Always include proper spacing between sections
- Use ## for main section headers
- Use **bold** for labels and important terms
- Use bullet points (•) for list items
- Each bullet on a new line
- Sub-bullets indented with proper hierarchy
- Blank line BEFORE and AFTER bulleted lists
- Keep paragraphs short (2-3 sentences max)

## Spacing Rules Summary

✅ DO:
- Use blank lines between major sections
- Use blank lines around bullet lists
- Use ## for section headers
- Use **bold** for important labels
- Keep bullet points concise
- Indent sub-bullets clearly

❌ DON'T:
- Bunch sentences together without spacing
- Use ALL CAPS for emphasis
- Mix formatting styles
- Have paragraphs longer than 3 sentences
- Skip blank lines between sections
